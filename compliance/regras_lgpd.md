# Regras de LGPD — Prospecção Weecode

Este projeto deve respeitar boas práticas de privacidade e proteção de dados.

## Permitido

- Usar dados públicos e institucionais
- Usar telefone geral da instituição
- Usar e-mail institucional publicado no site
- Usar informações de páginas oficiais
- Registrar fonte e data de coleta
- Usar contatos profissionais publicados de forma pública

## Proibido

- Coletar dados de pacientes
- Coletar dados sensíveis
- Usar base vazada
- Usar base comprada sem origem clara
- Usar e-mail pessoal
- Usar telefone pessoal sem contexto profissional
- Fazer disparo automático
- Enviar mensagens sem revisão humana

## Regras de abordagem

Toda abordagem deve:
- Ser profissional
- Ter contexto B2B
- Ser revisada por humano
- Permitir opt-out
- Evitar insistência excessiva
- Não prometer resultado sem diagnóstico
